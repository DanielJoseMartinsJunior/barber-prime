# Barber Prime 💈

Projeto desenvolvido com:

- React JS
- Vite
- TailwindCSS

## Estrutura

```bash
src/
 ├── assets/
 ├── components/
 ├── sections/
 ├── App.jsx
 ├── main.jsx
 └── index.css
```

## Organização

- components → componentes reutilizáveis
- sections → seções da página

## Estilo

- Tema premium
- Preto + dourado
- Responsivo
- Design moderno

## Rodar projeto

```bash
npm install
npm run dev
```
